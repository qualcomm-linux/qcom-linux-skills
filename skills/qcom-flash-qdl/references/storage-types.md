# Storage Types by Device

Default storage configurations for supported platforms. Storage type is baked
into prebuilt images and cannot be changed without rebuilding from source.

| Device         | Storage Type  |
|----------------|---------------|
| QCS6490        | UFS           |
| IQ-9075-EVK    | UFS           |
| IQ-8275-EVK    | UFS           |
| IQ-615-EVK     | EMMC          |
| IQ-X7181-EVK   | UFS/SPINOR    |
| IQ-X5121-EVK   | UFS/SPINOR    |
