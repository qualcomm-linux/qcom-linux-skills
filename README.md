# qli-skills

Catalog of Agent Skill files to be used with Qualcomm Linux.

## Branches

**main**: Primary development branch. Contributors should develop submissions based on this branch, and submit pull requests to this branch.

## Installation Instructions

Skills available under `skills` folder, to be copied over your favorite agent skills directory.

## Usage

Use your favorite agent to call one of the named skills from this project.
